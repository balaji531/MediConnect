📌 Overview

This AI Telemedicine Platform is a full-stack web application that enables secure virtual healthcare.
It provides AI-powered symptom checking, patient management, and real-time video consultations, all through a responsive and intuitive dashboard.

✨ Key Features

🩺 AI Symptom Checker

👥 Patient Management & Records

📅 Appointment Scheduling

🎥 Video Consultations

📊 Dashboard Analytics & Reports

📱 Fully Responsive UI

🛠 Tech Stack

| Layer        | Technology Used |
|-------------|-----------------|
| Frontend    | React, Tailwind CSS|
| Backend     | Node (Express) |
| Database    | MongoDB |
| Symptom Checker | Python |

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/balaji531/Telemedicine.git
cd Telemedicine
```
---

### 2️⃣ Setup Backup

```bash
cd backend
npm install
Configure .env
npm run dev
```
---

### 3️⃣ Setup Symptom Checker

```bash
cd Ai-Symptom-Chcker
cd backend
python -m venv venv
venv\scripts\activate
pip install -r requirements.txt
python app.py
```
---

### 5️⃣ Run the Application

```bash
cd frontend
npm install
npm run dev
```
Open in browser:
```bash
http://127.0.0.1:5173
```
